# Компоненты создания комнаты

Набор компонентов для создания и настройки размеров комнаты с 3D предпросмотром в реальном времени.

## Компоненты

### RoomCreator

Форма для ввода размеров комнаты с валидацией и предпросмотром.

**Функциональность:**
- Ввод ширины, высоты и глубины комнаты
- Валидация размеров (от 1 до 20 метров)
- Предпросмотр площади и объема в реальном времени
- Интеграция с Zustand store
- Кнопки применения и сброса

**Использование:**
```tsx
import { RoomCreator } from '@/components/room'

<RoomCreator 
  onRoomCreated={(dimensions) => console.log(dimensions)} 
/>
```

### RoomInfo

Отображение информации о размерах комнаты и расчетных параметров.

**Функциональность:**
- Отображение основных размеров
- Расчет площади пола, стен, объема
- Рекомендации на основе размеров
- Автоматическое обновление при изменении размеров

**Использование:**
```tsx
import { RoomInfo } from '@/components/room'

<RoomInfo />
```

### RoomSetup

Главный компонент, объединяющий создание комнаты и 3D предпросмотр.

**Функциональность:**
- Форма создания комнаты
- 3D предпросмотр в реальном времени
- Информация о комнате
- Кнопки действий

**Использование:**
```tsx
import { RoomSetup } from '@/components/room'

<RoomSetup 
  onComplete={(dimensions) => {
    // Переход к следующему этапу
  }} 
/>
```

## Интеграция с Store

Все компоненты интегрированы с Zustand store (`useRoomStore`):

```typescript
const { 
  roomDimensions,      // Текущие размеры комнаты
  setRoomDimensions    // Установка новых размеров
} = useRoomStore()
```

## Валидация

Размеры комнаты валидируются по следующим правилам:
- Минимальный размер: 1 метр
- Максимальный размер: 20 метров
- Только положительные числа
- Шаг изменения: 0.1 метра

## 3D Предпросмотр

3D сцена автоматически обновляется при изменении размеров:
- Отображение пола, стен и каркаса комнаты
- Адаптивная камера и сетка
- Полупрозрачные стены для лучшей видимости
- Wireframe для понимания размеров

## Требования

Соответствует требованиям из спецификации:
- ✅ 1.1 - Создание новой комнаты
- ✅ 1.2 - Ввод размеров (длина, ширина, высота)
- ✅ 1.3 - 3D-модель пустой комнаты
- ✅ 1.4 - Обновление в реальном времени
