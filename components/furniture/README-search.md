# Система поиска и фильтрации мебели

## Обзор

Реализована комплексная система поиска и фильтрации предметов мебели с поддержкой множественных критериев и интуитивным пользовательским интерфейсом.

## Основные возможности

### 1. Быстрый поиск
- **Поиск по названию**: Поиск предметов по названию в реальном времени
- **Поиск по цвету**: Поддержка поиска по цвету предмета
- **Автодополнение**: Мгновенные результаты при вводе

### 2. Расширенные фильтры
- **Фильтр по цене**: Диапазон цен с слайдером и числовыми полями
- **Фильтр по цвету**: Выбор из доступных цветов
- **Фильтр по стилю**: Фильтрация по стилям интерьера
- **Фильтр по размеру**: Категории размеров (маленький, средний, большой, очень большой)

### 3. Размерные категории
Предметы автоматически классифицируются по размеру на основе объема:
- **Маленький** (≤ 0.5 м³): Декор, подушки, небольшие аксессуары
- **Средний** (0.5-2.0 м³): Кресла, тумбы, небольшие столы
- **Большой** (2.0-5.0 м³): Диваны, обеденные столы, шкафы
- **Очень большой** (> 5.0 м³): Большие шкафы, кухонные гарнитуры

## Компоненты

### FurnitureLibrary
Основной компонент каталога с интегрированным поиском и фильтрацией.

**Возможности:**
- Быстрый поиск в шапке
- Переключение между категориями
- Расширенные фильтры
- Отображение результатов с количеством найденных предметов

### SearchFilters
Компонент расширенных фильтров с полным набором опций.

**Возможности:**
- Поиск по тексту
- Слайдер диапазона цен
- Фильтры по цвету, стилю и размеру
- Отображение активных фильтров с возможностью быстрого удаления
- Сброс всех фильтров

### Slider
Пользовательский компонент слайдера для выбора диапазона цен.

## API функции

### searchFurnitureAdvanced
Основная функция расширенного поиска с поддержкой всех фильтров.

```typescript
searchFurnitureAdvanced(
  query: string,
  filters: {
    categories?: FurnitureCategory[]
    minPrice?: number
    maxPrice?: number
    colors?: string[]
    styles?: RoomStyle[]
    sizeCategories?: SizeCategory[]
  }
): FurnitureItem[]
```

### getItemSizeCategory
Определяет размерную категорию предмета на основе его объема.

```typescript
getItemSizeCategory(item: FurnitureItem): SizeCategory
```

### getFurnitureBySizeCategories
Фильтрует предметы по размерным категориям.

```typescript
getFurnitureBySizeCategories(categories: SizeCategory[]): FurnitureItem[]
```

## Использование

### Быстрый поиск
```tsx
// Поиск выполняется автоматически при вводе
<Input
  placeholder="Быстрый поиск предметов..."
  value={searchQuery}
  onChange={(e) => handleSearch(e.target.value)}
/>
```

### Расширенные фильтры
```tsx
<SearchFilters
  filters={filters}
  onFiltersChange={handleFiltersChange}
  availableColors={availableColors}
  maxBudget={budget}
  isVisible={showFilters}
  onToggleVisibility={() => setShowFilters(!showFilters)}
/>
```

## Интеграция с бюджетом

Система фильтрации интегрирована с бюджетным планированием:
- Максимальная цена фильтра автоматически ограничивается текущим бюджетом
- При изменении бюджета фильтры обновляются автоматически
- Предупреждения о превышении бюджета учитываются в результатах поиска

## Производительность

- **Мемоизация**: Результаты поиска кешируются с помощью useMemo
- **Дебаунсинг**: Поиск выполняется с задержкой для снижения нагрузки
- **Ленивая загрузка**: Фильтры загружаются только при необходимости

## Тестирование

Создан компонент `SearchTest` для проверки всех функций:
- Тестирование поиска по названию
- Проверка всех типов фильтров
- Статистика по размерным категориям
- Комбинированные фильтры
