# Каталог мебели

Компоненты для отображения и управления каталогом мебели в приложении "Создатель Уютных Комнат".

## Компоненты

### FurnitureLibrary
Основной компонент каталога мебели с полным функционалом:
- Переключение между категориями
- Поиск по названию и цвету
- Фильтрация по цене, цвету и стилю
- Настройка видимости категорий
- Интеграция с системой бюджета

### FurnitureGrid
Сетка для отображения предметов мебели:
- Адаптивная сетка (2-4 колонки)
- Состояние загрузки
- Обработка пустых результатов

### FurnitureCard
Карточка отдельного предмета мебели:
- Изображение с fallback
- Информация о цене и размерах
- Кнопка добавления с проверкой бюджета
- Бейджи стилей

### CategoryTabs
Вкладки для переключения между категориями:
- Иконки категорий
- Счетчики предметов
- Настраиваемая видимость

### CategorySettings
Настройки видимости категорий:
- Включение/выключение категорий
- Быстрые действия (показать/скрыть все)
- Защита от скрытия всех категорий

## База данных мебели

Файл `lib/data/furniture-database.ts` содержит:
- 20+ предметов мебели с российскими ценами
- 6 категорий: мебель, текстиль, декор, освещение, растения, техника
- Функции для фильтрации и поиска
- Статистика по категориям

## Интеграция с системой

### Zustand Store
- Управление выбранной категорией
- Проверка бюджетных ограничений
- Добавление предметов в комнату
- Уведомления о превышении бюджета

### Типы данных
- `FurnitureItem` - предмет мебели
- `FurnitureCategory` - категории
- `RoomStyle` - стили интерьера

## Использование

```tsx
import { FurnitureLibrary } from '@/components/furniture'

export default function MyPage() {
  return (
    <div className="h-screen">
      <FurnitureLibrary />
    </div>
  )
}
```

## Особенности

1. **Бюджетный контроль**: Предметы блокируются при превышении лимита
2. **Адаптивность**: Работает на всех размерах экранов
3. **Производительность**: Ленивая загрузка и оптимизированный рендеринг
4. **Доступность**: Поддержка клавиатуры и screen readers
5. **Локализация**: Все тексты на русском языке
