# Project Brief: Room Designer (Создатель Уютных Комнат)

## Цель проекта
Создание веб-приложения для 3D дизайна интерьера с интеллектуальным управлением бюджетом и AI-рекомендациями.

## Основные задачи
1. Предоставить пользователям инструмент для визуализации дизайна комнаты в 3D
2. Помочь планировать бюджет на обустройство интерьера
3. Предложить AI-рекомендации по подбору мебели и оптимизации расходов
4. Интегрировать реальные данные о мебели с российских маркетплейсов

## Целевая аудитория
- Люди, планирующие ремонт или обустройство жилья
- Дизайнеры интерьера
- Пользователи, желающие визуализировать идеи перед покупкой мебели

## Ключевые особенности
- 3D визуализация в реальном времени
- Управление бюджетом с предупреждениями
- Каталог мебели с реальными ценами
- AI-рекомендации по стилю и бюджету
- Поиск и фильтрация предметов интерьера
- Сохранение и экспорт проектов

## Ограничения
- Бесплатные или условно-бесплатные решения
- Фокус на российский рынок (Wildberries, Ozon, Hoff)
- Работа в браузере без установки
