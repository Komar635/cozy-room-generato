# Project Brief

## Основная информация

- **Название проекта**: Оцифровщик реальности в 3D (Cozy Room Generator)
- **Тип**: Веб-приложение (SaaS)
- **Целевая платформа**: Web (Next.js)
- **Текущая версия**: 0.1.x

## Цели проекта

Создание веб-приложения для превращения фотографий мебели в интерактивные 3D-модели с возможностью модификации и интеграции в интерьеры.

## Границы проекта (Scope)

### Включено
- Загрузка фотографий и их обработка через AI (Gemini)
- Генерация 3D-моделей из фотографий
- Визуализация в WebGL (Three.js)
- Базовая аутентификация пользователей
- Управление проектами пользователей
- Хранение данных в Supabase (PostgreSQL)

### Не включено (Out of Scope)
- Мобильные приложения
- AR/VR режимы
- Продвинутая физика
- Маркетплейс моделей
- Коллаборация в реальном времени

## Ожидаемые результаты

### Для MVP
1. Регистрация/вход пользователей
2. Загрузка изображений
3. Генерация 3D-моделей
4. Просмотр и вращение моделей
5. Сохранение проектов

## Технические ограничения

- Supabase Free Tier лимиты
- Gemini API rate limits
- Размер файлов < 10MB
