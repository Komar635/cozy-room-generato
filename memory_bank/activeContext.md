# Active Context

## Текущий фокус

Работа над функционалом генерации 3D-моделей и интеграцией с Gemini API.

## Активные решения

- Использование Supabase для хранения данных
- NextAuth.js для аутентификации
- React Three Fiber для 3D-визуализации
- Gemini API для AI-генерации описаний

## Приоритеты

1. Стабилизация генерации 3D-моделей
2. Оптимизация производительности
3. Добавление новых функций модификации

## Известные неопределенности

- Точный формат вывода Gemini API
- Пределы бесплатного tier
