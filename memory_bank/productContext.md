# Product Context

## Проблемы пользователей

### 1. Сложность визуализации
Пользователи не могут представить, как будет выглядеть комната с новой мебелью до покупки.

### 2. Превышение бюджета
Часто при обустройстве интерьера люди тратят больше запланированного из-за отсутствия контроля.

### 3. Несоответствие стилей
Сложно подобрать мебель, которая гармонично сочетается по стилю и цвету.

### 4. Разрозненность информации
Цены и характеристики мебели разбросаны по разным маркетплейсам.

## Решение

### 3D Визуализация
- Создание виртуальной комнаты с точными размерами
- Размещение мебели в 3D пространстве
- Просмотр результата с разных углов

### Бюджетное планирование
- Установка лимита бюджета
- Отслеживание расходов в реальном времени
- Предупреждения при приближении к лимиту
- Предложения по оптимизации

### AI-помощник
- Рекомендации по стилю интерьера
- Подбор мебели по бюджету
- Предложения альтернатив при превышении бюджета

### Единый каталог
- Агрегация данных с Wildberries, Ozon, Hoff
- Сравнение цен
- Фильтрация по параметрам

## Ценностное предложение
"Спланируйте идеальный интерьер в 3D, не выходя за рамки бюджета"
