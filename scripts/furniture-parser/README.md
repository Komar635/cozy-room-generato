# 🛋️ Парсер мебели для Cloud.ru

Автоматический парсер цен на мебель с российских маркетплейсов (Wildberries, Ozon, Hoff) с сохранением в Cloud.ru PostgreSQL.

## 🚀 Возможности

- **Автоматический парсинг** с Wildberries, Ozon, Hoff
- **Мониторинг цен** - отслеживание min/avg/max цен
- **Категоризация** товаров по типам мебели
- **Сохранение в Cloud.ru** PostgreSQL базу данных
- **Планировщик** - автоматический запуск по расписанию
- **История цен** - отслеживание изменений стоимости
- **API готовность** - данные готовы для использования в веб-приложении

## 📦 Установка

```bash
cd scripts/furniture-parser
npm install
```

## ⚙️ Настройка

1. **Скопируйте файл конфигурации:**
```bash
cp .env.example .env
```

2. **Настройте подключение к Cloud.ru:**
```env
CLOUD_DB_HOST=your-cloud-ru-host
CLOUD_DB_PORT=5432
CLOUD_DB_NAME=furniture_db
CLOUD_DB_USER=your-username
CLOUD_DB_PASSWORD=your-password
```

3. **Создайте базу данных:**
```bash
# Подключитесь к PostgreSQL и выполните:
psql -h your-host -U your-user -d postgres -f src/database/schema.sql
```

## 📊 Структура данных

### Основные таблицы:

**furniture_items** - основная таблица мебели
- Название, категория, бренд
- Размеры (ширина, высота, глубина)
- Цвета, материалы, стили
- Min/avg/max цены
- Изображения и описания

**price_sources** - источники цен
- Цены с разных маркетплейсов
- Рейтинги и отзывы
- Наличие товара

**price_history** - история изменения цен
- Отслеживание динамики цен
- Аналитика по периодам

## 🔌 Интеграция с Next.js приложением

После запуска парсера данные будут доступны через API:

```javascript
// lib/services/furniture-api.js
const { query } = require('./database');

export async function getFurnitureItems(filters = {}) {
  const result = await query(`
    SELECT 
      fi.*,
      fc.name as category_name,
      fb.name as brand_name,
      (fi.price_min / 100.0) as price_min_rub,
      (fi.price_avg / 100.0) as price_avg_rub,
      (fi.price_max / 100.0) as price_max_rub
    FROM furniture_items fi
    LEFT JOIN furniture_categories fc ON fi.category_id = fc.id
    LEFT JOIN furniture_brands fb ON fi.brand_id = fb.id
    WHERE fi.is_active = true
    ORDER BY fi.updated_at DESC
    LIMIT 100
  `);
  
  return result.rows;
}
```

## 🚨 Обработка ошибок

Парсер устойчив к ошибкам:
- Автоматические повторы при сбоях
- Логирование всех ошибок
- Продолжение работы при недоступности одного источника
- Graceful shutdown при остановке
