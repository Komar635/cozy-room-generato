# ИИ Сервисы в проекте "Создатель Уютных Комнат"

## 🤖 Что такое "RoomGPT API" в нашем проекте?

**Короткий ответ**: "RoomGPT API" - это наш собственный интеграционный слой, который объединяет несколько реальных AI сервисов для создания умных рекомендаций по дизайну интерьера.

## 🔍 Почему не используем оригинальный RoomGPT?

Оригинальный RoomGPT (https://roomgpt.io) - это веб-приложение, а **не API сервис**. Он не предоставляет публичный API для интеграции в другие проекты.

## 🛠️ Наше решение: Гибридная архитектура

Мы создали собственную реализацию с аналогичным функционалом:

```
"RoomGPT API" = Replicate API + OpenAI API + Локальные алгоритмы
```

### Что каждый сервис делает:

1. **Replicate API** 🎨
   - Генерирует изображения интерьера
   - Модели: Stable Diffusion, ControlNet
   - Стоимость: $10 бесплатно

2. **OpenAI API** 🧠  
   - Анализирует комнату и рекомендует мебель
   - Модель: GPT-3.5-turbo
   - Стоимость: $5 бесплатно

3. **Локальные алгоритмы** 💻
   - Работают без интернета
   - Расчет бюджета, размещение мебели
   - Fallback когда API недоступны

## ✅ Преимущества нашего подхода:

- **Работает без API ключей** - базовый функционал доступен всегда
- **Экономично** - используем бесплатные кредиты
- **Надежно** - если один сервис не работает, используем другой
- **Гибко** - можем добавить любые новые AI сервисы

## 🚀 Как настроить (опционально):

1. Получить API ключи (бесплатно):
   - Replicate: https://replicate.com/
   - OpenAI: https://platform.openai.com/

2. Добавить в `.env.local`:
   ```
   REPLICATE_API_TOKEN=ваш_токен
   OPENAI_API_KEY=ваш_ключ
   ```

3. Перезапустить приложение

## 🎯 Результат:

Пользователь получает полноценный ИИ-помощник для дизайна интерьера, который:
- Рекомендует мебель по стилю и бюджету
- Генерирует изображения комнат
- Оптимизирует расходы
- Работает даже без внешних API

**Вывод**: Мы не "оставили" RoomGPT API - мы создали собственную, более гибкую реализацию с аналогичным функционалом! 🎉
