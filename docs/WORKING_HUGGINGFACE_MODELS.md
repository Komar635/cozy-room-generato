# 🔍 Рабочие HuggingFace модели для генерации изображений

## ⚠️ Проблема с устаревшими ссылками

Многие старые модели Stable Diffusion на HuggingFace больше не работают или были перемещены.

## 🎯 Проверенные рабочие модели (2024)

### 1. **CompVis/stable-diffusion-v1-4** ✅
- **Ссылка:** https://huggingface.co/CompVis/stable-diffusion-v1-4
- **API:** `https://api-inference.huggingface.co/models/CompVis/stable-diffusion-v1-4`
- **Статус:** Работает стабильно

### 2. **stabilityai/stable-diffusion-2-1** ✅
- **Ссылка:** https://huggingface.co/stabilityai/stable-diffusion-2-1
- **API:** `https://api-inference.huggingface.co/models/stabilityai/stable-diffusion-2-1`
- **Статус:** Официальная модель Stability AI

### 3. **dreamlike-art/dreamlike-diffusion-1.0** ✅
- **Ссылка:** https://huggingface.co/dreamlike-art/dreamlike-diffusion-1.0
- **API:** `https://api-inference.huggingface.co/modМнеels/dreamlike-art/dreamlike-diffusion-1.0`
- **Статус:** Специально для интерьеров

### 4. **Lykon/DreamShaper** ✅
- **Ссылка:** https://huggingface.co/Lykon/DreamShaper
- **API:** `https://api-inference.huggingface.co/models/Lykon/DreamShaper`
- **Статус:** Популярная модель для дизайна

## 🚀 Как проверить модель:

1. Откройте страницу модели на HuggingFace
2. Найдите кнопку "Deploy" → "Inference API"
3. Если есть - модель работает через API
4. Если нет - модель не поддерживает Inference API
