# Интеграция RoomGPT API - Руководство по настройке

## Обзор

Проект интегрирован с RoomGPT API для предоставления умных рекомендаций по дизайну интерьера с адаптацией под российский рынок и цены в рублях.

## Архитектура интеграции

### Гибридный подход
- **Основной**: RoomGPT API (OpenAI + Replicate) для максимального качества
- **Fallback**: Локальный ИИ на основе правил - всегда работает без интернета
- **Адаптация**: Все цены автоматически конвертируются в рубли

## Настройка API ключей

### 1. Replicate API (Рекомендуется)
```bash
# Получить: https://replicate.com/ → Account → API tokens
# Бесплатно: $10 кредитов (~500 изображений)
REPLICATE_API_TOKEN=r8_your_replicate_token_here
```

**Возможности:**
- Генерация изображений интерьера
- Высокое качество результатов
- Стабильная работа

### 2. OpenAI API (Для рекомендаций)
```bash
# Получить: https://platform.openai.com/api-keys
# Бесплатно: $5 кредитов (3 месяца)
OPENAI_API_KEY=sk-your_openai_key_here
```

**Возможности:**
- Умные рекомендации мебели
- Анализ и оптимизация бюджета
- Контекстуальные предложения

### 3. HuggingFace API (Альтернатива)
```bash
# Получить: https://huggingface.co/settings/tokens
# Бесплатно: ограниченное использование
HUGGINGFACE_API_KEY=hf_your_huggingface_token_here
```

**Статус:** ⚠️ Частично работает (некоторые модели недоступны)

## Адаптация под российский рынок

### Настройки валюты и локализации
```bash
# Минимальный бюджет для рекомендаций (в рублях)
MIN_BUDGET_RUB=10000

# Минимальный бюджет для дизайна комнаты (в рублях)  
MIN_DESIGN_BUDGET_RUB=15000

# Валюта для отображения цен
CURRENCY=RUB

# Локаль для форматирования цен
LOCALE=ru-RU
```

### Автоматическая адаптация цен
- Все цены автоматически округляются до целых рублей
- Форматирование в российском стиле: "45 000 ₽"
- Минимальные пороги адаптированы под российский рынок
- Рекомендации учитывают российские бренды и поставщиков

## Fallback стратегия

### Локальный ИИ (всегда работает)
Если внешние API недоступны, система автоматически переключается на локальные алгоритмы:

- **Рекомендации**: На основе правил дизайна интерьера
- **Оптимизация**: Математические алгоритмы поиска замен
- **Планировка**: Умная расстановка мебели по размерам комнаты
- **Цены**: Адаптированы под российский рынок

### Преимущества гибридного подхода
1. **Надежность**: Приложение работает даже без интернета
2. **Качество**: Лучшие результаты при наличии API
3. **Экономия**: Локальный ИИ не тратит API кредиты
4. **Скорость**: Мгновенные ответы от локальных алгоритмов

## Мониторинг и отладка

### Проверка статуса API
```typescript
import { AIApiService } from '@/lib/services/ai-api'

const status = await AIApiService.checkAIStatus()
console.log('Статус ИИ сервисов:', status)
```

### Логирование
Все API вызовы логируются в консоль с указанием источника:
- `source: "openai"` - OpenAI API
- `source: "replicate"` - Replicate API  
- `source: "local"` - Локальный ИИ
- `fallback: true` - Использован fallback

### Обработка ошибок
```typescript
try {
  const recommendations = await AIApiService.getRecommendations(params)
} catch (error) {
  // Система автоматически переключится на локальный ИИ
  console.log('Используется локальный ИИ из-за ошибки API')
}
```

## Безопасность

### Защита API ключей
- Все ключи хранятся в переменных окружения
- Никогда не передаются на клиент
- Проверка наличия ключей перед вызовами API

### Rate Limiting
- Автоматическое ограничение частоты запросов
- Graceful degradation при превышении лимитов
- Переключение на локальный ИИ при блокировке

## Стоимость использования

### Бесплатные лимиты
- **OpenAI**: $5 кредитов (3 месяца) = ~1000 рекомендаций
- **Replicate**: $10 кредитов = ~500 изображений
- **Локальный ИИ**: Безлимитно и бесплатно

### Оптимизация расходов
1. Используйте локальный ИИ для базовых задач
2. Внешние API только для сложных запросов
3. Кешируйте результаты рекомендаций
4. Настройте разумные лимиты запросов

## Поддержка и обновления

### Обновление API
Система спроектирована для легкого добавления новых API:
1. Добавьте новый сервис в `lib/services/`
2. Интегрируйте в `AIRecommendationService`
3. Обновите fallback логику

### Мониторинг производительности
- Отслеживайте время ответа API
- Мониторьте использование кредитов
- Анализируйте качество рекомендаций

Система готова к работе как с внешними API, так и в полностью автономном режиме!
