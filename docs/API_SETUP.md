## 🔑 Где получить API ключи

### 1. OpenAI API (для анализа и рекомендаций)

**Назначение:** Анализ стилей интерьера, генерация рекомендаций по мебели

**Как получить:**
1. Зарегистрируйтесь на https://platform.openai.com/
2. Перейдите в раздел API Keys
3. Создайте новый ключ
4. Скопируйте ключ (он показывается только один раз!)

**Стоимость:** 
- $5 бесплатных кредитов при регистрации
- Далее pay-as-you-go (примерно $0.002 за 1000 токенов для GPT-3.5)

**Переменная окружения:**
```env
OPENAI_API_KEY=sk-...
```

### 2. Replicate API (для генерации изображений)

**Назначение:** Генерация изображений интерьера на основе описания

**Как получить:**
1. Зарегистрируйтесь на https://replicate.com/
2. Перейдите в Account Settings → API Tokens
3. Создайте новый токен
