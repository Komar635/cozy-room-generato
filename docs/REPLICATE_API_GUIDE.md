# 🎯 Replicate API - Рабочее решение для генерации изображений

## ✅ Почему Replicate?

- **Стабильно работает** - нет проблем с устаревшими API
- **Бесплатные кредиты** - $10 при регистрации
- **Много моделей** - специально для дизайна интерьера
- **Простая интеграция** - один API для всех моделей

## 🚀 Быстрая настройка (2 минуты)

### 1. Регистрация
1. Перейдите на https://replicate.com/
2. Нажмите "Sign up" 
3. Зарегистрируйтесь через GitHub или email

### 2. Получение токена
1. После регистрации перейдите в "Account"
2. Найдите "API tokens" в меню слева
3. Нажмите "Create token"
4. Введите название (например, "Room Designer")
5. Скопируйте токен (начинается с `r8_`)

### 3. Настройка в проекте
```bash
# Создайте .env.local в корне проекта:
echo "REPLICATE_API_TOKEN=r8_ваш_токен_здесь" > .env.local

# Перезапустите приложение:
pnpm dev
```
