## 📋 Пошаговая настройка:

### 1. Регистрация (30 секунд)
1. Откройте https://replicate.com/
2. Нажмите "Sign up" 
3. Войдите через GitHub (быстрее всего)

### 2. Получение токена (30 секунд)
1. После входа нажмите на аватар → "Account"
2. В левом меню найдите "API tokens"
3. Нажмите "Create token"
4. Скопируйте токен (начинается с `r8_`)

### 3. Настройка проекта (30 секунд)
```bash
# Создайте .env.local в корне проекта:
echo "REPLICATE_API_TOKEN=r8_ваш_токен_здесь" > .env.local

# Перезапустите приложение:
pnpm dev
```

### 4. Проверка (30 секунд)
1. Откройте http://localhost:3000/setup
2. Нажмите "Проверить" - должно показать ✅
3. Перейдите на http://localhost:3000/api-demo
4. Попробуйте "Получить AI рекомендации"
