# Система стилей интерьера

## Обзор

Реализована комплексная система стилей интерьера с поддержкой 5 основных стилей, автоматической расстановкой мебели, интеграцией с ИИ-рекомендациями и анализом соответствия стилю.

## Поддерживаемые стили

### 1. Скандинавский (Scandinavian)
- **Цвета**: Светлые тона (#F5F5F5, #E8DCC4)
- **Материалы**: Натуральное дерево, ткань, металл
- **Характеристики**: Высокая яркость (0.9), низкий контраст (0.3), теплота (0.7)
- **Бюджет**: от 25,000₽ (оптимально 70,000₽)

### 2. Лофт (Loft)
- **Цвета**: Темные тона (#4A4A4A, #333333)
- **Материалы**: Металл, кирпич, кожа
- **Характеристики**: Низкая яркость (0.3), высокий контраст (0.8), прохлада (0.4)
- **Бюджет**: от 35,000₽ (оптимально 90,000₽)

### 3. Классика (Classic)
- **Цвета**: Теплые тона (#E3DAC9, #5D4037)
- **Материалы**: Дерево, ткань, золото
- **Характеристики**: Средняя яркость (0.6), высокая роскошь (0.9), теплота (0.8)
- **Бюджет**: от 50,000₽ (оптимально 120,000₽)

### 4. Модерн (Modern)
- **Цвета**: Нейтральные тона (#FFFFFF, #D3D3D3)
- **Материалы**: Стекло, металл, пластик
- **Характеристики**: Высокая яркость (0.8), высокий контраст (0.7), прохлада (0.3)
- **Бюджет**: от 30,000₽ (оптимально 80,000₽)

### 5. Минимализм (Minimalist)
- **Цвета**: Очень светлые тона (#FAFAFA, #F0F0F0)
- **Материалы**: Бетон, дерево, сталь
- **Характеристики**: Максимальная яркость (0.9), низкий контраст (0.2), низкая роскошь (0.3)
- **Бюджет**: от 20,000₽ (оптимально 60,000₽)

## Компоненты системы

### StyleSelector
Компонент выбора стиля с:
- Визуальными карточками стилей
- Информацией о бюджете
- Ключевыми словами стиля
- Цветовыми схемами
- Интеграцией с ИИ-рекомендациями

### StyleAnalyzer
Компонент анализа соответствия стилю:
- Процентный показатель соответствия
- Список несоответствующих предметов
- Рекомендации по улучшению
- Кнопка автоматического улучшения

### StyleTemplates
Система шаблонов с:
- Цветовыми схемами
- Настройками освещения
- Базовой мебелью
- Характеристиками стиля
- Приоритетами предметов

## API и сервисы

### StyleUtils
Утилитарные функции:
- `getRecommendedBudget()` - расчет рекомендуемого бюджета
- `getPriorityFurniture()` - получение приоритетной мебели
- `isColorCompatible()` - проверка совместимости цветов
- `getStyleCharacteristics()` - характеристики стиля

### LocalAIService (улучшенный)
Локальный ИИ с поддержкой стилей:
- Стиль-специфичные рекомендации
- Учет существующей мебели
- Оптимизация с сохранением стиля
- Анализ соответствия

### AIRecommendationService (расширенный)
Интеграция с внешними ИИ:
- Адаптация под российские цены
- Учет стилевых предпочтений
- Fallback на локальный ИИ
- Анализ стилевой консистентности

## Интеграция с приложением

### RoomStore (обновленный)
Добавлены новые действия:
- `applyStyleTemplate()` - применение шаблона стиля
- `analyzeStyleConsistency()` - анализ соответствия
- `getStyleRecommendations()` - получение рекомендаций по бюджету

### Автоматическая расстановка
При выборе стиля:
1. Применяется цветовая схема
2. Настраивается освещение
3. Размещается базовая мебель
4. Учитываются размеры комнаты
5. Соблюдается бюджет

## Тестирование

Создан комплексный набор тестов:
- Валидация шаблонов стилей
- Проверка утилитарных функций
- Тестирование ИИ-сервисов
- Интеграционные тесты

## Использование

```typescript
// Применение стиля
const { applyStyleTemplate } = useRoomStore()
applyStyleTemplate(RoomStyle.SCANDINAVIAN)

// Анализ соответствия
const { analyzeStyleConsistency } = useRoomStore()
const analysis = analyzeStyleConsistency()

// Получение рекомендаций
const recommendations = await AIRecommendationService.getFurnitureRecommendations({
  roomDimensions: { width: 4, height: 3, depth: 4 },
  style: RoomStyle.MODERN,
  budget: 50000
})
```

## Особенности реализации

1. **Адаптивность**: Все цены адаптированы под российский рынок
2. **Fallback**: При недоступности внешних API используется локальный ИИ
3. **Гибкость**: Возможность применения стиля с сохранением существующей мебели
4. **Производительность**: Локальные алгоритмы работают мгновенно
5. **Расширяемость**: Легко добавить новые стили и характеристики
