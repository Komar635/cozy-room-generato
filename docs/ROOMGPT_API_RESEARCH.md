## 📋 Требуемый функционал от RoomGPT:

1. **Генерация дизайна комнаты**
   - Вход: размеры комнаты, стиль, бюджет
   - Выход: расстановка мебели, цветовая схема

2. **Рекомендации по мебели**
   - Вход: параметры комнаты
   - Выход: список подходящей мебели

3. **Оптимизация бюджета**
   - Вход: текущая мебель, целевой бюджет
   - Выход: альтернативы для экономии

4. **Генерация изображений**
   - Вход: описание интерьера
   - Выход: изображение комнаты
