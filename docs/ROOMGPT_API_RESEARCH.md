# RoomGPT API Integration

## 🎯 Что такое "RoomGPT API" в нашем проекте

**RoomGPT API** - это не отдельный сервис, а **интеграционный слой** который объединяет несколько реальных AI сервисов для создания функциональности, аналогичной оригинальному RoomGPT проекту.

### Архитектура RoomGPT API:

```
RoomGPTApiService
├── Replicate API (генерация изображений интерьера)
├── OpenAI API (умные рекомендации мебели)
├── HuggingFace API (бесплатная альтернатива)
└── Local Algorithms (fallback когда API недоступны)
```

## 🔧 Реальные API которые мы используем:

### 1. **Replicate API** (основной для изображений)
- **Что делает**: Генерирует изображения интерьера по описанию
- **Модели**: `jagilley/controlnet-hough`, `stability-ai/stable-diffusion`
- **Стоимость**: $10 бесплатных кредитов (~500 изображений)
- **Регистрация**: https://replicate.com/

### 2. **OpenAI API** (для рекомендаций)
- **Что делает**: Анализирует параметры комнаты и рекомендует мебель
- **Модель**: `gpt-3.5-turbo`
- **Стоимость**: $5 бесплатных кредитов (3 месяца)
- **Регистрация**: https://platform.openai.com/

### 3. **HuggingFace API** (бесплатная альтернатива)
- **Статус**: ❌ Временно не работает (ошибка 410)
- **Что делало**: Бесплатная генерация изображений
- **Альтернатива**: Используем Replicate API

## 📋 Функционал "RoomGPT API":

### ✅ Реализовано:
1. **Генерация дизайна комнаты** - через Replicate + локальные алгоритмы
2. **Рекомендации по мебели** - через OpenAI + база данных мебели
3. **Оптимизация бюджета** - локальные алгоритмы
4. **Генерация изображений** - через Replicate API
5. **Fallback система** - работает без внешних API

### 🔄 Как это работает:

```typescript
// Пример: получение рекомендаций
const recommendations = await RoomGPTApiService.getFurnitureRecommendations({
  roomDimensions: { width: 4, height: 3, depth: 4 },
  style: 'scandinavian',
  budget: 100000
})

// Внутри RoomGPTApiService:
// 1. Пробует OpenAI API
// 2. Если не работает -> локальные рекомендации
// 3. Возвращает результат с пометкой источника
```

## 🚀 Настройка (2 минуты):

### Шаг 1: Получить API ключи
```bash
# Replicate (рекомендуется)
https://replicate.com/ → Sign up → API tokens

# OpenAI (опционально)
https://platform.openai.com/api-keys → Create key
```

### Шаг 2: Добавить в .env.local
```env
REPLICATE_API_TOKEN=r8_ваш_токен_здесь
OPENAI_API_KEY=sk-ваш_ключ_здесь
```

### Шаг 3: Перезапустить приложение
```bash
pnpm run dev
```

## ✅ Преимущества нашего подхода:

1. **Работает без API** - локальные алгоритмы как fallback
2. **Гибкость** - можно использовать любые AI сервисы
3. **Экономия** - используем бесплатные кредиты разных сервисов
4. **Надежность** - если один API не работает, используем другой

## 🔍 Почему не оригинальный RoomGPT API:

Оригинальный RoomGPT (https://github.com/Nutlope/roomGPT) - это **веб-приложение**, а не API сервис. Он использует те же технологии (Replicate, OpenAI), что и мы, но не предоставляет публичный API.

Поэтому мы создали собственную реализацию с аналогичным функционалом.
