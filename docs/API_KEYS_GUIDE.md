# 🔑 Полное руководство по получению API ключей

## 🎯 Какие API нужны для полного функционала

### 1. 🖼️ **Replicate API** (Приоритет #1)
**Для чего:** Генерация изображений интерьера, дизайн комнат

#### 📋 Пошаговая инструкция:
1. **Перейдите на** https://replicate.com/
2. **Нажмите "Sign up"** в правом верхнем углу
3. **Зарегистрируйтесь** через GitHub, Google или email
4. **Подтвердите email** если требуется
5. **Перейдите в настройки**: Нажмите на аватар → "Account"
6. **Найдите "API tokens"** в левом меню
7. **Создайте токен**: "Create token" → введите название → "Create"
8. **Скопируйте токен** (показывается только один раз!)

#### 💰 Стоимость:
- **Бесплатно**: $10 кредитов при регистрации
- **Тарифы**: Pay-per-use, ~$0.01-0.10 за генерацию
- **Модели**: Разные цены для разных моделей

#### ⚙️ Настройка:
```env
REPLICATE_API_TOKEN=r8_ваш_токен_здесь
```

---

### 2. 🤖 **OpenAI API** (Приоритет #2)
**Для чего:** Умные рекомендации мебели, анализ стилей

#### 📋 Пошаговая инструкция:
1. **Перейдите на** https://platform.openai.com/
2. **Нажмите "Sign up"** или "Log in"
3. **Зарегистрируйтесь** через email или Google
4. **Подтвердите номер телефона** (обязательно!)
5. **Перейдите в API Keys**: https://platform.openai.com/api-keys
6. **Создайте ключ**: "Create new secret key"
7. **Введите название** (например, "Room Designer")
8. **Скопируйте ключ** (начинается с sk-)

#### 💰 Стоимость:
- **Бесплатно**: $5 кредитов при регистрации (действуют 3 месяца)
- **GPT-3.5-turbo**: $0.002 за 1K токенов (~500 слов)
- **GPT-4**: $0.03 за 1K токенов
- **Примерно**: 1 рекомендация = $0.01-0.05

#### ⚙️ Настройка:
```env
OPENAI_API_KEY=sk-ваш_ключ_здесь
```

---

### 3. 🆓 **Hugging Face API** (Бесплатная альтернатива)
**Для чего:** Бесплатные AI модели, альтернатива OpenAI

#### 📋 Пошаговая инструкция:
1. **Перейдите на** https://huggingface.co/
2. **Нажмите "Sign Up"**
3. **Зарегистрируйтесь** через email или GitHub
4. **Подтвердите email**
5. **Перейдите в настройки**: Аватар → "Settings"
6. **Найдите "Access Tokens"** в левом меню
7. **Создайте токен**: "New token" → "Write" права → "Generate"
8. **Скопируйте токен** (начинается с hf_)

#### 💰 Стоимость:
- **Полностью бесплатно** для большинства моделей
- **Лимиты**: Есть rate limits, но щедрые
- **Модели**: Тысячи бесплатных моделей

#### ⚙️ Настройка:
```env
HUGGINGFACE_API_KEY=hf_ваш_токен_здесь
```

## 🚀 Быстрая настройка (5 минут)

### Минимальная настройка (только Replicate):
```bash
# 1. Получите Replicate токен (см. выше)
# 2. Создайте .env.local в корне проекта:
echo "REPLICATE_API_TOKEN=r8_ваш_токен" > .env.local

# 3. Перезапустите приложение:
pnpm dev
```

### Полная настройка (все API):
```bash
# Создайте .env.local со всеми ключами:
cat > .env.local << EOF
REPLICATE_API_TOKEN=r8_ваш_replicate_токен
OPENAI_API_KEY=sk-ваш_openai_ключ
HUGGINGFACE_API_KEY=hf_ваш_huggingface_токен
EOF

# Перезапустите:
pnpm dev
```

## 🛠️ Решение проблем

### ❌ "API ключ не работает"
1. **Проверьте формат**: 
   - Replicate: `r8_...`
   - OpenAI: `sk-...`
   - HuggingFace: `hf_...`
2. **Перезапустите сервер** после добавления ключей
3. **Проверьте баланс** в личном кабинете API

### ❌ "Превышен лимит"
1. **OpenAI**: Добавьте карту в биллинг
2. **Replicate**: Пополните баланс
3. **HuggingFace**: Подождите сброса лимитов

### ❌ "Ошибка сети"
1. **Проверьте интернет**
2. **Попробуйте VPN** (некоторые API блокируют регионы)
3. **Проверьте firewall**

## 💡 Рекомендации по использованию

### 🎯 Для начала:
1. **Начните с Replicate** - самый простой в настройке
2. **Добавьте OpenAI** для лучших рекомендаций
3. **HuggingFace** как резерв

### 💰 Для экономии:
1. **Используйте кэширование** результатов
2. **Ограничьте количество запросов** в день
3. **Начните с бесплатных лимитов**

### 🚀 Для продакшена:
1. **Настройте все три API** для надежности
2. **Добавьте мониторинг** расходов
3. **Используйте fallback** на локальные алгоритмы
